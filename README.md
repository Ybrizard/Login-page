Este projeto consiste em uma página de login moderna construída utilizando somente HTML e CSS.  
O foco está na estruturação semântica, design responsivo e boa experiência do usuário.  
A interface foi desenvolvida como componente inicial para aplicações web que necessitam de autenticação, podendo ser facilmente integrada a back-ends ou sistemas maiores.
